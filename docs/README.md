# 🗳️ VoteIndia: Democratic Intelligence Platform

[![Vite](https://img.shields.io/badge/Vite-B73BFE?style=for-the-badge&logo=vite&logoColor=FFD62D)](https://vitejs.dev/)
[![Firebase](https://img.shields.io/badge/firebase-ffca28?style=for-the-badge&logo=firebase&logoColor=black)](https://firebase.google.com/)
[![Accessibility](https://img.shields.io/badge/Accessibility-100%25-green?style=for-the-badge)](https://www.w3.org/WAI/standards-guidelines/wcag/)

**VoteIndia** is a premium, AI-powered election command center designed to provide Indian citizens with verified, data-driven insights into the democratic process. From localized candidate profiles (KYC) to interactive voting explainers and a standalone Electoral Quiz, VoteIndia simplifies complex governance data into a stunning, accessible experience.

## ✨ Key Features

- 📟 **EVM/VVPAT Explainer**: High-fidelity visual guide to the voting process.
- 🔍 **Candidate KYC**: Instant search for assets, education, and criminal records across constituencies.
- 📈 **Regional Trends**: Real-time ticker of electoral shifts with deep-zoom insights.
- 🗳️ **Democratic Quiz**: Standalone 10-question challenge to test civic knowledge.
- 📅 **Google Services Sync**: Live localized clock and 1-click "Add to Calendar" reminders for election dates.
- 🤖 **VoteIndia AI**: Smart assistant for personalized electoral guidance.

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-repo/voteindia.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment**:
   Create a `.env` file and add your Firebase and Google Cloud keys:
   ```env
   VITE_FIREBASE_API_KEY=your_key
   VITE_GOOGLE_MAPS_API_KEY=your_key
   ```
4. **Launch Development Server**:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend**: Vite + Vanilla JS + Inter/Outfit Typography
- **Styles**: Modern CSS with Glassmorphism and Hardware Acceleration
- **Backend**: Firebase Auth & Firestore
- **External**: Google Maps Timezone API & Google Calendar Template API
- **Testing**: Vitest Suite for logic and data integrity
